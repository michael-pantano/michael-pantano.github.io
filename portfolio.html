MongoDB Graphing Dashboard via Streamlit:

This python program is able to search through every post in a database and return every unique tag/test name. From there, streamlit reads the text input of a tag name and the program searches through every post with that tag name and parses through the data and stores it in arrays. The program then graphs the data!

Virtual Dashboard to Display and Control Height of System:

This virtual dashboard reads the input of a distance sensor and displays it. When the height is set, the dashboard sends data to the height motors to increase or decrease the height until it reaches the set value.
